# [FEATURE SPEC] 업그레이드 시스템 개편
우선순위: HIGH | 복잡도: 복잡(~8h) | 기획일: 2026-03-14

## 개요
기존 3카테고리 12종 단순 업그레이드 → Tier 1-2 선형 + Tier 3 분기 + 시너지 시스템으로 개편.
총 약 22종 업그레이드 + 6개 시너지.

## 핵심 설계 원칙
1. Tier 1-2: 코인으로 즉시 구매 (선형)
2. Tier 3 분기: 웨이브 조건 달성 시 해금, 되돌릴 수 없음
3. 시너지: 특정 조합 보유 시 자동 발동 보너스
4. 구매 즉시 타워 시각적 변화 (색상 변화, 크기 소폭 증가)

## 구현 범위

### 수정 파일
- `src/config/upgradeConfig.js` — 전면 재작성 (Tier/분기/시너지 구조)
- `src/game/GameManager.js` — purchaseUpgrade, 시너지 계산 추가
- `src/game/TowerSystem.js` — SNIPER/GATLING 분기, POISON/FREEZE 특수 효과
- `src/components/UpgradePanel.jsx` — Tier 트리 UI, 분기 선택 확인 다이얼로그
- `src/components/Tower.jsx` — 업그레이드 티어별 시각 변화

## 시너지 시스템

시너지는 특정 업그레이드 조합 보유 시 자동 발동.
GameManager.js의 계산 후 towerStats에 반영.

| 시너지 ID | 이름 | 조건 | 효과 |
|----------|------|------|------|
| atk_mastery | 전사의 분노 | 공격 카테고리 4종 이상 | 공격력 +25% |
| def_mastery | 철벽 | 방어 카테고리 4종 이상 | 피해 무효화 확률 5% |
| util_mastery | 전략가 | 유틸 카테고리 4종 이상 | 코인 획득 +30% |
| blood_revenge | 피의 복수 | thorn + lifesteal 동시 | 반사 피해로 흡혈 발동 |
| ice_poison | 얼음 독 | freeze + poison 동시 | 빙결 적에게 독 피해×2 |
| sniper_poison | 맹독 저격 | sniper + poison 동시 | 단일 타겟에 독 효과 지속 시간×3 |

## UI 구성

```
UpgradePanel.jsx 구조:
┌──────────────────────────────┐
│  [공격] [방어] [유틸]  (3탭)  │
├──────────────────────────────┤
│                              │
│  Tier1●──●──●max             │
│             └── Tier2●──●max │
│                      └── [분기A] / [분기B]  (잠금 시 🔒 웨이브N)
│                              │
│  [선택된 업그레이드 설명]      │
│  비용: 💰 150  [구매]         │
│                              │
│  시너지: ⚡ 피의 복수 활성!    │  ← 달성 시 표시
└──────────────────────────────┘
```

- 분기 선택 시 확인 다이얼로그: "이 선택은 되돌릴 수 없습니다. [확인] [취소]"
- 잠긴 업그레이드: 반투명 + "웨이브 X 달성 후 해금"
- 시너지 활성화 토스트 메시지 (2초 표시)

## 타워 시각적 변화 (Tier별)
- Tier 1 업그레이드 1개 이상: 타워 외곽선 밝기 증가
- Tier 2 업그레이드 1개 이상: 타워 크기 +10%, 외곽선 더블링
- Tier 3 분기 선택: 색상 변화 (공격분기→주황, 방어분기→파랑, 유틸분기→초록)
