# Neon Tower Defense

미니멀한 네온 스타일의 타워 디펜스 게임

## 주요 기능

- 육각형 타워 자동 공격 시스템
- 웨이브 기반 적 생성 시스템 (웨이브마다 적 수/스탯 증가)
- 3가지 카테고리 업그레이드 (공격/방어/유틸리티)
- 네온 스타일 Canvas 렌더링
- 실시간 코인 획득 애니메이션 및 파티클 이펙트

## 실행 방법

```bash
npm install
npm start
```

## 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- Canvas API (게임 렌더링)
- React Hooks (상태 관리)
- Tailwind CSS

## 게임 조작

| 키 | 동작 |
|---|---|
| `ESC` | 업그레이드 패널 열기/닫기 |
| `Space` / `P` | 일시정지 |
| UPGRADE 버튼 | 업그레이드 패널 |

## 업그레이드 카테고리

### 공격 (Attack)
- 공격력 강화 - 기본 데미지 증가
- 공격 속도 - 초당 공격 횟수 증가
- 사거리 확장 - 공격 범위 증가

### 방어 (Defense)
- 최대 체력 - 체력 증가
- 체력 재생 - 초당 회복량 증가
- 방어율 - 받는 피해 % 감소
- 절대 방어 - 고정 피해 감소
- 가시 갑옷 - 반사 피해
- 흡혈 - 공격 시 체력 회복

### 유틸리티 (Utility)
- 코인 보너스 - 적 처치 보상 증가
- 다중 공격 - 동시 공격 대상 증가
- 둔화 효과 - 적 이동속도 감소

## 밸런스 설정

`src/config/gameConfig.js` 및 `src/config/upgradeConfig.js`에서 수정 가능
